# Notes-Full-Stack-Website
 Notes Website - Frontend, Backend & Database
# Frontend: 
  HTML, CSS, JS
# Backend:
  node, express
# Database:
   MongoDB, mongoose
# Other packages:
   jsonwebtoken, bcrypt, dotenv, express-fileupload
# .env
  Create .env file in backend directory
  Add these: 
           PORT = 3000
           DB_URL = [DATABASE_URL]
           USER_SECRET_KEY = [ANY_KEY]
